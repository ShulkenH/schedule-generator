# 日程表生成器

一个可自定义的日程表梗图生成器，支持上传图片、编辑文字、导出 PNG/SVG。

## ✨ 功能

- 📷 点击上传头像图片
- ✏️ 所有文字可直接编辑
- ➕ 添加/删除事件卡片
- 📥 导出高清 PNG 或矢量 SVG
- 📱 响应式布局

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 🌐 部署到 GitHub Pages

1. Fork 或 clone 此仓库
2. 修改 `vite.config.js` 中的 `base` 为你的仓库名
3. 在 GitHub 仓库设置中启用 Pages（Settings → Pages → Source 选择 GitHub Actions）
4. 推送代码到 `main` 分支，自动部署
